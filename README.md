# Paulina Oña - Archivo de Obra ꩜

Este es el repositorio del portafolio interactivo y archivo de obra de **Paulina Oña**, artista e investigadora. El proyecto utiliza una interfaz experimental basada en coordenadas espaciales y un atlas visual dinámico.

## 🚀 Despliegue en Vivo
Puedes visitar el sitio web en: [https://darg88.github.io/Portafolio/](https://darg88.github.io/Portafolio/)

---

## 🎨 Características del Proyecto

- **Navegación Dual:**
  - **Vista Mapa (꩜):** Una constelación interactiva de proyectos que orbitan alrededor del eje central de la artista. Permite rotación mediante el scroll y exploración mediante arrastre.
  - **Vista Mosaico (⊞):** Un atlas visual inspirado en el "Atlas Mnemosyne" de Aby Warburg, donde las obras se presentan como un muro de memorias organizado de forma orgánica.
- **Diseño Responsivo:** Adaptación completa para dispositivos móviles, laptops y pantallas de gran formato.
- **Interactividad Avanzada:** Sistema de "germinación" de conexiones mediante doble clic y efectos de parallax en tiempo real.
- **Bilingüe:** Soporte nativo para Español e Inglés con persistencia de selección.

## 🛠️ Stack Tecnológico

- **Frontend:** HTML5 semántico y CSS3 (Custom Properties, Flexbox, Grid).
- **Interactividad:** JavaScript Vanilla (ES6+) con arquitectura modular.
- **Optimización:**
  - Desplazamiento sincronizado mediante `requestAnimationFrame`.
  - Carga diferida de imágenes (*Lazy Loading*).
  - Gestión de datos centralizada en `data.js`.

## 📁 Estructura del Repositorio

- `index.html`: Estructura principal y contenedores de vistas.
- `style.css`: Diseño visual, animaciones y media queries.
- `script.js`: Motor de lógica interactiva y manipulación del DOM.
- `data.js`: Base de datos modular de proyectos y categorías.
- `/img`: Directorio de registros visuales (fotografías y afiches).
